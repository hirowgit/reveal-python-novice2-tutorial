<6,1>

ここでは、リスト型から要素を取り出す方法を学びます。

* リスト型で、要素番号（場所）を指定して、番号に対応する要素を取り出すことができます。
* ただし、**最初の要素は、１番はなく、０番であることに注意**してください。
* つまり、リストL=[1,2,3]が与えられた場合、０番目の要素が１で、１番目の要素が２で、２番目の要素が２になります。

* 尚、一番最後の要素は、L[-1]のように、-1番目と指定して取り出すことができます。

* リスト型には、.insert()メソッドがあり、要素番号（場所）を指定して、要素を挿入することができます。
* ただし、**要素番号は、０から始まることに注意**してください。
* つまり、リストL=[1,2,3]が与えられた場合、０番目に、要素4を入れたい場合、L.insert(０番目、要素4）、つまり、L.insert(0,4)と書き、リストLは[4,1,2,3]になります。

* .insert()は、数字でも、文字列でも指定の場所に、挿入できることは変わりません。

* ちなみに、sort(reverse=False)のように、反転プロパティをFalseにする（反転スイッチをOFF）ことで、()に何も指定しない場合と同様、小さい順から並べることができます。

1. [3,2,1]に対し、sortメソッドと適切な設定（プロパティ）をつかって、[3, 2, 1]に並べ替えて、ans1に入れます。

```text
ans1.sort(reverse=???)
# ans1= [3, 2, 1]
```

```python
ans1 = [1,2,3]
ans1.sort(reverse=True)
# ans1= [3, 2, 1]
```

--
<8,2>

2. [5,4,1,3,2]に対し、sortメソッドと適切な設定（プロパティ）をつかって、[5, 4, 3, 2, 1]に並べ替えて、ans2に入れます。
