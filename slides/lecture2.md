# L2M3S1

## リスト型の取り扱い

<3,1>

ここでは、リスト型の取り扱いを学びます。

* リスト型では、「,」で区切って要素を並べることでデータをひとまとめにして扱うことができます。
* 要素は、数字でも、文字または文字列でも問題ありません。
* リスト型では、左右の端を**[]の記号**で囲みます（ディクショナリ型の場合は、{}で囲みます）。

1. １から３までの数字を、リスト型としてまとめ、ans1に入れます。

```text
#Level 1:
ans1= ?1,2,3?
```

```python
ans1= [1,2,3]
```

--

<3,2>

2. １から４までの数字を、リスト型としてまとめ、ans2に入れます。

```text
#Level 2:
ans2= ?1,2,3,4?
```

```python
ans2= [1,2,3,4]
```

--

<3,3>

3. aからcまでの文字を、リスト型としてまとめ、ans3に入れます。

```text
#Level 3:
ans3= [?a?,?b?,?c?]
```

```python
ans3= ['a','b','c']
```

# L2M3S2

## 発展編：リスト型の結合（＋演算子）

<3,4>

* リスト型では、+演算子で結合し、複数の部分リストを一つのリストにすることができます。
* 要素は、数字でも、文字または文字列でも問題ありませんし、それらが混在していても問題ありません。
* 要素が文字、または文字列の場合は、「'」または「"」で要素を囲みます。

1. [1,2,3] と [4]を結合し、[1,2,3,4]のリスト型として、ans1に入れます。

```text
#Level 1:
ans1= [1,2,3] ? [4]
```

```python
ans1= [1,2,3] + [4]
```

<3,5>

2. [3,2,1] と [5,4]を結合し、[3,2,1,5,4]のリスト型として、ans2に入れます。

```text
#Level 2:
ans2= [3,2,1] ? [5,4]
```

```python
ans2= [3,2,1] + [5,4]
```

--

<3,6>

3. ['a'] と ['b','c'] と [1,2] と ['d']を結合し、['a', 'b', 'c', 1, 2, 'd']のリスト型として、ans3に入れます。

```text
#Level 3:
ans3= ['a'] ? ['b','c'] ? [1,2] ? ['d']
```

```python
ans3= ['a'] + ['b','c'] + [1,2] + ['d']
```

# L2M4S1

## リスト型とメソッド（その１：append）

<4,1>

ここでは、リスト型の取り扱いを学びます。

* リスト型では、+演算子の他に、.appendメソッド（関数の一種）を使って要素の追加やリストの結合を行うことができます。
* 関数には２種類あり、「abc（）」のように、()内に対象のオブジェクトを入れて計算処理する通常の関数と、「オブジェクト（object）の属性として参照される関数」として、object.abc()のように作用させる**メソッドと呼ばれる関数**があります。
* append（）の場合、()内に、要素でなく、リストを入れると、リストの中にリストが入る入れ子構造（階層リスト）ができることに注意します。
* したがって、階層リストにしたくない場合は、extend()メソッドを使うか、+演算子でリスト型を結合します。

1. .appendメソッドを用いて[1,2,3]と要素4を結合し、[1, 2, 3, 4]にして、ans1に入れます。

```text
#Level 1:
ans1 = [1,2,3]
ans1.append(??)
# ans1= [1, 2, 3, 4]
```

```python
ans1 = [1,2,3]
ans1.append(4)
# ans1= [1, 2, 3, 4]
```

--

<4,2>

2. .appendメソッドを用いて[3,2,1]と[5,4]を結合し、[3, 2, 1, [5, 4]]にして、ans2に入れます。

```text
ans2 = [3,2,1]
ans2.append(??)
# ans2= [3, 2, 1, [5, 4]]
```

```python
ans2 = [3,2,1]
ans2.append([5,4])
# ans2= [3, 2, 1, [5, 4]]
```

--

<4,3>

3. .appendメソッドを用いて['a']と['b','c','d']を結合し、['a', ['b', 'c', 'd']]にして、ans3に入れます。

```text
ans3 = ['a']
ans3.append(??)
# ans3= ['a', ['b', 'c', 'd']]
```

```python
ans3 = ['a']
ans3.append(['b','c','d'])
# ans3= ['a', ['b', 'c', 'd']]
```
